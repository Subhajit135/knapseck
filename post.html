Kharagpur Winter of Code|KWOC

Project Report

Posted by Subhajit Saha on January 7, 2018

ABOUT KWOC

Kharagpur Winter of Code is a 5-week long online programme for the students, who are new to open source software development. The programme not only helps students to get involved in open source, but also preps them for many open source summer programmes, Google Summer of Code being one of them.

PROJECT REPORT

Choosing the Project: To be honest this was the most toughest part as there was so many interesting projects that i want to work on. But after short listing i finally worked on two projects. Being a python developer it's is my privilege to contribute in the open-source communinity. So i participated in python currated list and my mentor for this project was Prateek Chanda and the project was CODE SLEEP PYTHON and also being a typical maths lover it was irrestible to ignore the projects related to algorithms so i also contributed to SMVDU-ALGOS which was mentored by Sushil khanchi | Shubham Goyal.

Progress Report: First i have started contributing to the project "CODE SLEEP PYTHON", it was a curated list of various python projects, i contributed some projects like Koch Snowflake - The Koch snowflake (also known as the Koch curve, Koch star, or Koch island) is a mathematical curve and one of the earliest fractal curves to have been described. It is based on the Koch curve. After that i contributed superellipse - A superellipse, also known as a Lamé curve after Gabriel Lamé, is a closed curve resembling the ellipse, retaining the geometric features of semi-major axis and semi-minor axis, and symmetry about them, but a different overall shape.

After that I got interested in the project named SMVDU-ALGOS which is basically another repo covering a curated list of algorithms where i first contributed COCKTAIL SHAKER SORT basically an sorting algorithm which i implemented in python. After that i have contributed A* algorithm - one of the smartest path finding algorithm based on heurestics out in the wild, which i also implemented in python.

Summary

It was great initiative to get started to open source contribution for me, along with also getting an rough idea about GSoC. Basically i was able to contribute 4 articles in two repositories.
